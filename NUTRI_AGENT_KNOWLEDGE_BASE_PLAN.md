# План доработки базы знаний Mimo / Nutri Intelligence Agent

## Решение

Mimo не поставляется с чужими профессиональными методичками. База знаний по умолчанию пустая: нутрициолог загружает собственные материалы или документы, на использование которых у него есть права.

Агент не должен проводить полноценный анализ клиента без базы знаний. Если источников нет, интерфейс обязан явно объяснить, что анализ невозможен: агенту не на что опираться при интерпретации, ограничениях и рекомендациях.

## Клиентский путь нутрициолога

1. Нутрициолог открывает кабинет и видит статус базы знаний.
2. Если база пуста, статус: `База не готова`.
3. Интерфейс объясняет, какие материалы нужны:
   - методика первичного разбора клиента;
   - правила интерпретации анкеты, дневника и файлов;
   - ограничения, red flags и запреты;
   - шаблон итогового отчета.
4. Нутрициолог может:
   - загрузить свои PDF;
   - открыть безопасное демо на вымышленных источниках;
   - скачать шаблоны для заполнения своей базы.
5. Диалог агента блокирует переход к анализу, пока база знаний пуста.

## Демо-контент

Демо должно показывать механику продукта, но не подменять профессиональную методологию.

Встроенное демо:
 - вымышленное;
 - помечено как `не для клиентов`;
 - не содержит чужих методичек;
 - не является медицинской или нутрициологической инструкцией.

## Правовая гигиена

В продукт нельзя включать купленные или полученные для личного использования методички третьих лиц. Даже переработанные версии могут оставаться производными материалами.

Для коммерческого использования подходят:
 - собственные авторские материалы;
 - материалы с явной лицензией на коммерческое использование;
 - публичные источники с разрешающей лицензией;
 - специально написанные демо-шаблоны без заимствования методологии.

## Реализовано в прототипе

 - Добавлен статус готовности базы знаний.
 - Пустая база знаний стала явным стоп-фактором.
 - Старый демо-набор заменен на безопасные вымышленные PDF-структуры.
 - Добавлены PDF-шаблоны для подготовки собственной базы.
 - Диалог агента больше не продолжает анализ без загруженных источников.
 - Старые демо-ссылки на PDF из `assets/methods` отфильтровываются при загрузке сохраненного состояния.
