# Nutri Mind

Прототип и спецификации AI-агента-нутрициолога (рабочее название проекта — *Mimo / Nutri Agent*).

> ⚠️ Приватный репозиторий. Содержит рабочие материалы продукта. Чужие методички,
> экспорты переписок, договоры и примеры клиентских данных намеренно исключены из
> публикации через [.gitignore](.gitignore).

## Структура

| Путь | Назначение |
|------|------------|
| [`nutri-agent/`](nutri-agent/) | Рабочий прототип (одностраничное HTML-приложение) и [`SPECIFICATION.md`](nutri-agent/SPECIFICATION.md) |
| [`NUTRI_AGENT_MVP_ARCHITECTURE.md`](NUTRI_AGENT_MVP_ARCHITECTURE.md) | Архитектура MVP |
| [`NUTRI_AGENT_CLIENT_MODEL_PLAN.md`](NUTRI_AGENT_CLIENT_MODEL_PLAN.md) | План модели клиентского кабинета |
| [`NUTRI_AGENT_KNOWLEDGE_BASE_PLAN.md`](NUTRI_AGENT_KNOWLEDGE_BASE_PLAN.md) | План базы знаний |
| [`NUTRI_AGENT_WORKING_PROTOTYPE_PLAN.md`](NUTRI_AGENT_WORKING_PROTOTYPE_PLAN.md) | План рабочего прототипа |
| [`NUTRI_AGENT_REWORK_PLAN.md`](NUTRI_AGENT_REWORK_PLAN.md) | План доработок |

## Что не входит в репозиторий

- `Nutri Agent исходники/` — чужие методички, договоры, примеры карт клиентов;
- `ChatExport_*/` — экспорты переписок;
- встроенные в прототип PDF-методички (`nutri-agent/assets/methods/*.pdf`).

Эти материалы хранятся только локально.

## Локальный запуск прототипа

Открыть [`nutri-agent/index.html`](nutri-agent/index.html) в браузере.
